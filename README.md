# 网上购物

---
## Spring_MVC、MyBatis、Redis、JQuery、Ajax、Maven
>* spring-mvc作为主体设计模式
>* MyBatis配合MyBatis plugin 插件使用，使得对数据库的查询便捷
>* redis数据缓存，用于模拟发送验证码
>* JQuery 使得JavaScript的编程更加便捷
>* Ajax 异步对页面中的部分内容进行更新（如首页获取购物车数量等）
>* Maven 用于管理项目依赖
